## Prepare 

### Linux Ubuntu 

1. cd tgbot
2. python3 -m venv venv
3. source venv\bin\activate
4. python3 -m pip install -r requirements.txt
5. python3 main.py

### Windows 10

1. cd tgbot
2. python -m venv venv
3. venv\Scripts\activate
4. python -m pip install -r requirements.txt
5. python main.py
